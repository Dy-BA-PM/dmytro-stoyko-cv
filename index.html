ERP-подібна система на Google Apps Script (GAS)

Що зроблено

  • • Єдина модель даних: довідники, коди, статуси, ролі, правила введення
  • • Рольовий доступ (реєстр користувачів) без правок коду під кожну зміну
  • • Журналювання змін / контроль якості даних / антидублювання
  • • Дашборди KPI/SLA: on-time, lead/cycle, WIP, прострочки, навантаження
  • • Інтеграційний контур (зокрема Telegram) для швидкої фіксації робіт

Модулі/контури

Projects — портфель/план, відповідальні, статуси, терміни
Production — черги/задачі, виконання, фіксація факту/часу, WIP
Warehouse — рух позицій/матеріалів, комплектність, звірки/інвентаризації
KPI/SLA — аналітика, контроль прострочок і якості даних
